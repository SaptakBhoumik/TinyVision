## üìÅ File Structure

- üîß **Model Weights**: Stored in `./model_weights/{model_name}.pt`
- üß† **Training Scripts**: Available in notebooks like `./model{i}.ipynb`, which include class-wise accuracy, loss, and training/validation accuracy curves for each model.
  - Example: `model1_0` and `model1_1` share the same architecture but are trained independently (different batch size or augmentations)

---

## üèãÔ∏è Training Details

- **Epochs**: 10 total
- **Optimizer**: AdamW (`Adam2`)
- **Learning Rate Schedule**:
  - Epochs 1‚Äì6: `0.001`
  - Epochs 7‚Äì9: `0.0001`
  - Remaining epochs: `0.00001`
- **Batch Size**: `40` (two models were trained with `16`)
- **Input Image Size**: `128 √ó 128`
- **Augmentation**: Horizontal flip ‚Äî every image has a flipped version, doubling the training set
